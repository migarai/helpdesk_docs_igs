# Devices Connectivity
The whole process of using **Teltonica** routers to have their **RMS**, Setting up **OpenVPN CloudConnexa** or **Ngrok** is to get the all the devices(RPI, FR, HAC etc) into a **Same Network**. In this process **IP Distribution** is important because <span style={{ color: "red" }}> to make sure one IP is only assigned once within a network.</span> Most of the time the devices using for system is manufactured by [Hikvision](https://www.hikvision.com/en/).
