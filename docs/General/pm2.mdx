# pm2

npm package: [https://www.npmjs.com/package/pm2](https://www.npmjs.com/package/pm2)

## pm2 commands
Before using commands `cd` to `v2Client` or `v2client`.  

* Get the **Logs**.
  ```bash
   pm2 logs
  ```
* Restart the service(**Resync**).
  ```bash
   pm2 restart all
  ```
* Stop the service
  ```bash
   pm2 stop all
  ```
* Start the service
  ```bash
   pm2 start ecosystem.config.js
  ```
